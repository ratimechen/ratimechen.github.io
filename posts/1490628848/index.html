把 Ubuntu 20.04 打造成 macOS 風格

發表於2021-05-03|Ubuntu
|字數總計:1.3k|閱讀時長:4分鐘

前置作業

安裝必裝工具

使用終端機開啟快捷鍵 Ctrl + Alt + T 來開啟終端機，並在終端機輸入指令

  1. sudo apt install gnome-tweak-tool gnome-shell-extensions chrome-gnome-shell

安裝四種擴充套件

使用瀏覽器打開 https://extensions.gnome.org/ 並點擊 Click here to install browser extension 進行瀏覽器擴充套件的安裝動作。

安裝完成把瀏覽器重開，並且再次使用瀏覽器打開 https://extensions.gnome.org/，在 Search for extensions... 欄位分別輸入 User Themes、Extensions 、Lock screen background、Dash to Dock 這四種擴充套件的名稱。

搜尋到的結果會顯示在下方，並點選尋結果以進入頁面進行安裝的動作

進入頁面點選 OFF 按鈕

跳出提示訊息，點選安裝來進行安裝的動作

剛才的 OFF 按鈕變成 ON 表示已經成功安裝 User Themes 了，而剩下的其他套件依此類推來進行安裝

下載主題、游標狀態、圖標、背景圖片

下載主題 (themes)

進入 WhiteSur Gtk Theme 主題包頁面之後，點選 WhiteSur-light.tar.xz 進行主題包的下載

下載游標狀態 (cursors)

進入 WhiteSur cursors 游標狀態包頁面之後，點選 WhiteSur-cursors.tar.xz 進行游標狀態包的下載

下載圖標 (icons)

進入 Mkos-Big-Sur 圖標包頁面之後，點選 Mkos-big-Sur.tar.xz 進行圖標包的下載

下載背景圖片

進入 mcOS BS for Cairo Dock 頁面之後，點選 macOS-BS-Walls.zip 進行背景圖片包的下載

把主題、游標狀態、圖標放入專屬資料夾內

設定顯示隱藏檔

點選桌面上的 testubuntu 資料夾兩下來進入 家目錄

進入 家目錄 後，點選視窗最小化 icon - 左側的 icon ☰，並把 顯示隱藏檔(H) 打勾

新增 .themes 和 .icons 資料夾

在 家目錄 內，按下 Shift+Ctrl+N 來進行新增資料夾，並在 Folder name 欄位中輸入 .themes 之後按下 Create 來完成新增 .themes 資料夾

在 家目錄 內，按下 Shift+Ctrl+N 來進行新增資料夾，並在 Folder name 欄位中輸入 .icons 之後按下 Create 來完成新增 .icons 資料夾

已經成功完成新增 .themes 和 .icons 資料夾

把主題、游標狀態、圖標分別放進 .themes 和 .icons 資料夾中

把下載好的 WhiteSur-light.tar.xz 壓縮檔進行解壓縮，並把解壓縮產生的資料夾 WhiteSur-light 放在 家目錄 下的 .themes 資料夾中

把下載好的 WhiteSur-cursors.tar.xz 壓縮檔進行解壓縮，並把解壓縮產生的資料夾 WhiteSur-cursors 放在 家目錄 下的 .icons 資料夾中

把下載好的 Mkos-big-Sur.tar.xz 壓縮檔進行解壓縮，並把解壓縮產生的三個資料夾 Mkos-Big-Sur、Mkos-Big-Sur-Night、Mkos-Big-Sur-Panel-white 放在 家目錄 下的 .icons 資料夾中

.icons 資料夾中有四個資料夾就表示放置完成了

設定外觀樣式

搜尋調校應用程式

點選 顯示應用程式

並在 Type to search 輸入 tweaks

點選 調校 來準備進行設定的動作

設定視窗大小控制鍵位置

在 Window Titlebars 的 Placement 設定為 Left，就會把調整視窗大小控制鍵的位置移到左側

設定外觀

  1. 在 外觀 的 應用程式 選擇 WhiteSur-light
  2. 在 外觀 的 游標 選擇 WhiteSur-cursors
  3. 在 外觀 的 圖示 選擇 Mkos-Big-Sur
  4. 在 外觀 的 Shell 選擇 WhiteSur-light
  5. 在 外觀 的 Image 可以自行選擇想要設定的桌面背景圖片

設定工作列位置

在 擴充套件 的 Dash to dock 點選 齒輪圖示，並把點選 齒輪圖示 之後打開的 Dash to Dock 視窗裡面的 Position on screen 改為 Bottom

工作列位置就變成在底下了

設定鎖定畫面背景

在 擴充套件 的 Lock screen background 點選 齒輪圖示，並把點選 齒輪圖示 之後打開的 Lock screen background 視窗裡面的 Browse 進行點選，並選擇想拿來設定為鎖定畫面背景的圖片

這樣就完成鎖定畫面背景的設定了

設定登入畫面背景

使用終端機開啟快捷鍵 Ctrl + Alt + T 來開啟終端機，並在終端機分別輸入指令

  1. git clone https://github.com/kp300/BACKGROUND-LOGIN-SCREEN.git
  2. cd BACKGROUND-LOGIN-SCREEN
  3. sudo bash login-background.sh ubuntu
  1. 在開啟的終端畫面 Choose a number 1/2: 輸入 1 並按下 Enter
  2. 在開啟的終端畫面 Open the folder and Select Image? 輸入 y 並按下 Enter

之後會彈出要你選擇圖片的視窗畫面，點選要選擇的圖片檔案，選好後按下 OK 來完成設定

看到終端機顯示的資訊就表示成功完成設定，並且可以登出系統來看設定的結果

登出系統就會看到背景圖片換成你設定的圖片了
