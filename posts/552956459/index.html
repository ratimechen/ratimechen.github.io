使用 GitHub 來存放部落格的圖片

發表於2021-04-26|部落格圖片存放空間
|字數總計:483|閱讀時長:1分鐘

前置作業

註冊或登入你的 GitHub 帳號

註冊或登入你的 GitHub 帳號

下載 PicGo

下載 PicGo 並選擇你的作業系統對應的檔案進行下載和安裝

GitHub 設置儲存庫

新增儲存庫

點選 New repository 來新增儲存庫

創建一個新的儲存庫

  1. Repository name 欄位填上你要為儲存庫命名的名稱
  2. Description (optional) 欄位填上你要為儲存庫描述的名稱 ( 此欄位為選擇性填寫可以不填 )
  3. 選擇 Public 的方式
  4. 都填選完畢就可以按下 Create repository 來完成創建儲存庫

看到此畫面表示儲存庫創建成功

創建 token

點選此連結來創建 token

點選 Generate new token 來進入 token 設置頁面

  1. Note 欄位設置為你的儲存庫名稱即可

  2. 記得要勾選 repo

確認 Note 欄位已填選和勾選 repo，並且按下 Generate token 來完成創建 token

記得要把創建出來的 token 記下來

PicGo 設置

GitHub 圖床設置

  1. 設定倉庫名: 填寫你的 github 用戶名稱 / 填寫你的 github 上要存放圖片的儲存庫名稱
  2. 設定分支名: 填入master
  3. 設定Token: 填入 github 設置儲存庫中創建出來的 token
  4. 指定存儲入徑: img/test/
  5. 設定自訂義域名: https://cdn.jsdelivr.net/gh/填寫你的 github 上要存放圖片的儲存庫名稱 (使用 jsdeliver 來加速圖片存取速度)
  6. 確認欄位都有填寫就能按下確定來完成設置

準備進行圖片上傳

將你要上傳的圖片拖到虛線範圍，或是點擊進行上傳

如果上傳成功桌面右下角會顯示此圖示

查看 GitHub 儲存庫

看到剛才上傳的圖片表示上傳成功

準備在部落格中使用圖片吧

這個 圖片連結 就可以拿來在部落格中使用了
